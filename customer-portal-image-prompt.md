- Large text input box with placeholder: "Describe your brand, colors, logo, and vision..."

COLOR PALETTE:
- Background: #1a1a2e, #16213e (dark navy gradients)

TECHNICAL DETAILS:
- Modern web UI, desktop view (1920x1080 resolution)

IMPORTANT:

---

## Key Elements to Visualize

1. **Simplicity**: One text input, like talking to ChatGPT
2. **AI Power**: Shows AI generating multiple design options instantly
3. **Visual Results**: 3D product mockups, not just sketches
4. **Manufacturability**: Designs that can actually be produced
5. **Professional**: Enterprise-ready, not a toy or prototype
6. **Speed**: Instant generation implied by the interface flow
